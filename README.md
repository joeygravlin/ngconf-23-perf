# Push-Based Angular Web Performance Workshop

## Installation Instructions

**System Requirements**

* `node 18`
* `npm 8`

**Clone and install**

```bash
git clone https://github.com/push-based/<repo-name>.git

cd <repo-name>
npm install

# (optional) if the step before didn't work, please try the following
npm install --force
```

**Run the application**

```bash
npm run start
```
